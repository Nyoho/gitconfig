.gitconfig or .config/git/config
==========

This is a repository for `~/.config/git/config` file.

How to install
--------------

Just `clone` it into `~/.config/git`.


If you like use `.gitconfig`, make a symbolic link by using

```
./make-symbolic-link.sh
```
